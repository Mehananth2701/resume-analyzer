# Resume Analyzer & Job Matcher

This is a Python-based project that analyzes resumes and matches them with job descriptions.

## Features
- Extracts skills from resume
- Compares with job requirements
- Calculates match score
- Suggests missing skills

## Technologies Used
- Python
- NLTK
- Scikit-learn

## How to Run
1. Install required libraries:
   pip install nltk scikit-learn PyPDF2

2. Run the project:
   python main.py

3. Enter resume and job description file paths

## Sample Output
- Match Score
- Missing Skills
- Suggestions
